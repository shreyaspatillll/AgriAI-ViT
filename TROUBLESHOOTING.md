## Installation Issues

### Problem: pip install fails with dependency conflicts
**Symptoms:**
```
ERROR: pip's dependency resolver does not currently take into account all the packages that are installed.
ERROR: Cannot install torch and torchvision
```

**Solutions:**
1. **Create fresh virtual environment:**
   ```bash
      python -m venv venv_agriai
         source venv_agriai/bin/activate  # Linux/Mac
            venv_agriai\Scripts\activate     # Windows
               ```

               2. **Install PyTorch first:**
                  ```bash
                     # CUDA 11.8
                        pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

                              # CPU only
                                 pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
                                    ```

                                    3. **Then install other requirements:**
                                       ```bash
                                          pip install -r requirements.txt
                                             ```

                                             ### Problem: ImportError for specific packages
                                             **Symptoms:**
                                             ```
                                             ImportError: No module named 'timm'
                                             ImportError: No module named 'transformers'
                                             ```

                                             **Solutions:**
                                             ```bash
                                             pip install timm transformers comet-ml --upgrade
                                             ```

                                             ### Problem: Version conflicts with existing packages
                                             **Solution:**
                                             ```bash
                                             pip install --upgrade --force-reinstall -r requirements.txt
                                             ```

                                             ### Problem: Kaggle dataset download fails
                                             **Symptoms:**
                                             - "401 Unauthorized" error
                                             - "kaggle.json not found"

                                             **Solutions:**
                                             1. **Setup Kaggle API credentials:**
                                                ```bash
                                                   # Create .kaggle directory
                                                      mkdir -p ~/.kaggle

                                                            # Copy kaggle.json
                                                               cp path/to/kaggle.json ~/.kaggle/
                                                                  chmod 600 ~/.kaggle/kaggle.json
                                                                     ```

                                                                     2. **Verify credentials:**
                                                                        ```bash
                                                                           kaggle datasets list
                                                                              ```

                                                                              3. **Manual download alternative:**
                                                                                 - Go to Kaggle dataset page
                                                                                    - Click "Download" button
                                                                                       - Extract to `data/raw/` directory

                                                                                                                                                                                             ## Training Issues
                                                                                                                                                                                             
                                                                                                                                                                                             ### Problem: Training crashes immediately
                                                                                                                                                                                             **Symptoms:**
                                                                                                                                                                                             ```
                                                                                                                                                                                             RuntimeError: CUDA out of memory
                                                                                                                                                                                             RuntimeError: Expected tensor to be on same device
                                                                                                                                                                                             ```
                                                                                                                                                                                             
                                                                                                                                                                                             **Solutions:**
                                                                                                                                                                                             1. **Reduce batch size:**
                                                                                                                                                                                                ```python
                                                                                                                                                                                                   # In config.yaml
                                                                                                                                                                                                      training:
                                                                                                                                                                                                           batch_size: 16  # Reduce from 32
                                                                                                                                                                                                              ```
                                                                                                                                                                                                              
                                                                                                                                                                                                              2. **Enable gradient accumulation:**
                                                                                                                                                                                                                 ```python
                                                                                                                                                                                                                    accumulation_steps = 4
                                                                                                                                                                                                                       for i, (images, labels) in enumerate(train_loader):
                                                                                                                                                                                                                              outputs = model(images)
                                                                                                                                                                                                                                     loss = criterion(outputs, labels) / accumulation_steps
                                                                                                                                                                                                                                            loss.backward()
                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                          if (i + 1) % accumulation_steps == 0:
                                                                                                                                                                                                                                                                     optimizer.step()
                                                                                                                                                                                                                                                                                optimizer.zero_grad()
                                                                                                                                                                                                                                                                                   ```
                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                   3. **Use mixed precision training:**
                                                                                                                                                                                                                                                                                      ```python
                                                                                                                                                                                                                                                                                         from torch.cuda.amp import autocast, GradScaler
                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                               scaler = GradScaler()
                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                     with autocast():
                                                                                                                                                                                                                                                                                                            outputs = model(images)
                                                                                                                                                                                                                                                                                                                   loss = criterion(outputs, labels)
                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                         scaler.scale(loss).backward()
                                                                                                                                                                                                                                                                                                                            scaler.step(optimizer)
                                                                                                                                                                                                                                                                                                                               scaler.update()
                                                                                                                                                                                                                                                                                                                                  ```
                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                  ### Problem: Loss becomes NaN
                                                                                                                                                                                                                                                                                                                                  **Symptoms:**
                                                                                                                                                                                                                                                                                                                                  ```
                                                                                                                                                                                                                                                                                                                                  Epoch 5: train_loss = nan, val_loss = nan
                                                                                                                                                                                                                                                                                                                                  ```
                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                  **Solutions:**
                                                                                                                                                                                                                                                                                                                                  1. **Reduce learning rate:**
                                                                                                                                                                                                                                                                                                                                     ```python
                                                                                                                                                                                                                                                                                                                                        optimizer = AdamW(model.parameters(), lr=1e-5)  # Instead of 3e-5
                                                                                                                                                                                                                                                                                                                                           ```
                                                                                                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                                                           2. **Add gradient clipping:**
                                                                                                                                                                                                                                                                                                                                              ```python
                                                                                                                                                                                                                                                                                                                                                 torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
                                                                                                                                                                                                                                                                                                                                                    ```
                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                    3. **Check for invalid data:**
                                                                                                                                                                                                                                                                                                                                                       ```python
                                                                                                                                                                                                                                                                                                                                                          # Add data validation
                                                                                                                                                                                                                                                                                                                                                             assert not torch.isnan(images).any()
                                                                                                                                                                                                                                                                                                                                                                assert not torch.isinf(images).any()
                                                                                                                                                                                                                                                                                                                                                                   ```
                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                   ### Problem: Model not learning (loss not decreasing)
                                                                                                                                                                                                                                                                                                                                                                   **Symptoms:**
                                                                                                                                                                                                                                                                                                                                                                   - Loss stays constant
                                                                                                                                                                                                                                                                                                                                                                   - Accuracy stuck at random chance
                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                   **Solutions:**
                                                                                                                                                                                                                                                                                                                                                                   1. **Check learning rate:**
                                                                                                                                                                                                                                                                                                                                                                      ```python
                                                                                                                                                                                                                                                                                                                                                                         # Try different learning rates
                                                                                                                                                                                                                                                                                                                                                                            for lr in [1e-6, 1e-5, 3e-5, 1e-4]:
                                                                                                                                                                                                                                                                                                                                                                                   optimizer = AdamW(model.parameters(), lr=lr)
                                                                                                                                                                                                                                                                                                                                                                                      ```
                                                                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                                                                      2. **Verify data loading:**
                                                                                                                                                                                                                                                                                                                                                                                         ```python
                                                                                                                                                                                                                                                                                                                                                                                            # Visualize batch
                                                                                                                                                                                                                                                                                                                                                                                               import matplotlib.pyplot as plt

                                                                                                                                                                                                                                                                                                                                                                                                                    # Make sure model is trainable
                                                                                                                                                                                                                                                                                                                                                                                                                       for param in model.parameters():

                                                                                                                                                                                                                                                                                                                                                                                                                                                   2. **Add dropout:**

                                                                                                                                                                                                                                                                                                                                                                                                                                                                             3. **Use weight decay:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                ```python
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   optimizer = AdamW(model.parameters(), lr=3e-5, weight_decay=0.01)

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            if val_loss > best_val_loss:

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   ## Model Performance Issues
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   ### Problem: Low accuracy on specific classes
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   **Symptoms:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   - Overall accuracy: 85%
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   - Specific class accuracy: 30%
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   **Solutions:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   1. **Collect more data for weak classes**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   2. **Apply class-specific augmentation**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   3. **Use focal loss:**
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      ```python
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         from torch.nn import functional as F
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               def focal_loss(outputs, targets, alpha=0.25, gamma=2.0):
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      ce_loss = F.cross_entropy(outputs, targets, reduction='none')
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             pt = torch.exp(-ce_loss)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    focal_loss = alpha * (1 - pt) ** gamma * ce_loss
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           return focal_loss.mean()
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              ```
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              ### Problem: Confusion between similar diseases
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              **Solutions:**
